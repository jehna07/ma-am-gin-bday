🌹 Happy Birthday Animation — Gift for Auntie 🌹
=================================================

FILES:
  index.html        → The full birthday animation (open this!)
  happy-birthday.mp3 → Happy Birthday melody (plays in-browser)
  happy-birthday.wav → Original WAV audio file

HOW TO USE:
  1. Keep all 3 files in the same folder.
  2. Open index.html in any browser (Chrome, Firefox, Safari).
  3. On Android phone: tap the screen to begin (required for audio).
  4. Watch the full animation — roses grow, get cut, form a bouquet,
     then a fireworks celebration plays with birthday music!

FEATURES:
  ✦ 25 animated roses growing, blooming, and forming a bouquet
  ✦ Golden scissors cutting each stem
  ✦ Luxury gift wrap & ribbon reveal
  ✦ Fireworks celebration with Happy Birthday melody
  ✦ 15 languages: Filipino, Spanish, French, Japanese, Korean,
     Chinese, Arabic, Portuguese, German, Hindi, Russian, Thai,
     Italian, Greek, English
  ✦ 🎵 Music toggle button (top-right corner)
  ✦ ↺ Replay button at end
  ✦ Fully optimized for Android phones — no lag, no blur

Made with ❤️ using HTML5 Canvas + Web Audio API.
